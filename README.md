# active-seismic-control
Active Seismic Vibration Control System 🏢⚡
This project features an Active Mass Damper (AMD) system designed to enhance structural stability during seismic events. By integrating Raspberry Pi with real-time sensors and motorized tracks, the system actively dissipates energy to prevent building collapse.

Achievement
Awarded "Best Seismic Design" in the Civil Engineering Design Practice course at National Taiwan University (NTU).
